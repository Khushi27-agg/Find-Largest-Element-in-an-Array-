# Find-Largest-Element-in-an-Array-
Day 10 of my daily coding challenge focuses on finding the largest element in an array using C++. The program iterates through the array, compares each element, and stores the maximum value using a function-based approach.

# Features

Finds the largest element in a numeric array
Uses a separate function for logic
Efficient single-pass comparison
Clean and readable implementation

# Concepts Used

Arrays
Functions
for loop
Conditional statements

# Technology
Language: C++
